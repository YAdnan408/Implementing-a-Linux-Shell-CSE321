# Linux Shell (CSE321 Project)

A simple custom Linux shell implementation in C, designed to demonstrate core concepts like process creation, command execution, piping, and redirection.

## Getting Started

### Prerequisites
*   GCC compiler
*   A Linux-based system

### Build and Run

To compile and run the shell:

```bash
gcc -o myshell myshell.c
./myshell
```

### Features
*   Command execution
*   I/O Redirection (<, >, >>)
*   Piping (|)
*   Sequential/Logical command operators (; , &&)
*   History tracking
